# Mitigator Project

Mitigator is a comprehensive trading bot designed to execute and analyze trades based on a predefined strategy. This project includes features for backtesting, live trading, and detailed analysis of trading performance.

## Features

- **Automated Trading**: Execute trades automatically based on predefined strategies.
- **Backtesting**: Test trading strategies on historical data.
- **Performance Analysis**: Generate detailed reports and visualizations of trading performance.
- **Trade History Management**: Store and manage trade history in JSON format.
